A series of Sublime Text 3 themes inspired by a journey into outerspace.
Let your code be illuminated by the shimmer of a planet.

Installing Interstellar

Via Package Control

Theme is listed as Theme - Interstellar in Will Bond's Sublime Package Control.

Manual

  1. Download and unzip the repo
  2. Copy the folder into Packages directory, which you can find using the menu item Preferences -> Browse Packages... in Sublime Text

Setup

Activate the UI theme and color scheme by modifying your user preferences file, which you can find using the menu item Preferences -> Settings - User in Sublime Text.

Example settings:

{
  "theme": "Interstellar.sublime-theme",
  "color_scheme": "Packages/interstellar-package/Moon.tmTheme"
}
